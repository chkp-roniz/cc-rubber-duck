# Rubber Duck — Cross-Model Review

A rubber-duck skill is available for cross-model second opinions via Codex CLI.

**Invoke it automatically** (read the full skill for details) when:

- You finish a plan touching 3+ files or non-trivial tradeoffs
- You write/edit 100+ lines of meaningful changes across multiple files
- You complete a test suite and are about to declare done
- You've retried the same fix 2+ times
- You're about to do something destructive (migrations, deletions, prod config)

**Invoke on demand** when the user says "rubber duck", "second opinion",
"critique this", "what am I missing", or types `/rubber-duck`.

To run: read and follow `.claude/skills/rubber-duck/SKILL.md`.
